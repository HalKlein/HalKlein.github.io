# 风


<!--more-->

文/张曼娟

------

![](https://i.loli.net/2019/12/09/bzdorXp9shRVJTq.jpg)

<center><font size=4>
    走过芦荻丛生的草原,
    <br/>
	我总觉得听见了悠扬的笛声。
    <br/>
	是风笛。你说。
    <br/>
	风是有声音的,
    <br/>
	只是我们没有留心细听。
    <br/>
	于是,你教我听,
    <br/>
	风借着山岳、草叶或是屋檐上的风铃,
    <br/>
	流转出低吟浅唱或是澎湃激昂。
    <br/>
</font></center>
